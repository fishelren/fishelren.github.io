JavaScript中的变量提升

发表于 2017-04-28 |

JavaScript中除了有函数声明提升，还有变量提升。但变量提升只对var有效，对let是无效的。而且这种变量提升是一种“不完全”提升，为什么说是“不完全”提升呢？来看下面的例子：

console.log(x); //undefined
var x=1;

可以看到打印x的时候并没有报未定义（ReferenceError）的错，声明的确是提升了；但是在定义语句前引用var定义的变量得到的值是undefined，说明变量赋值未提升，这是“未完全”

console.log(x); //ReferenceError
let x=1;

可见，let就不存在变量的提升，必须在用let定义变量后再使用变量

这只是概念上的一些理解，实际应用中我们还是“定义在前，使用在后”吧！
